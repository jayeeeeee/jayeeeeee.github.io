### Environment
Windows 10

### Shortcuts

#### Basic Editing
-   [ Ctrl ] + [ F2 ] : Select all occurrences of current word

#### Rich Languages Editing
-   [ Shift ] + [ Alt ] + [ F ] : Auto format

#### Display
-   [ Ctrl ] + [ Shift ] + [ V ] : Preview markdown
