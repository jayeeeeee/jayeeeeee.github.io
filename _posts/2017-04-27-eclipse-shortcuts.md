Neon 

### Shortcuts

#### File
    [ Ctrl ] + [ W ] : Close current file

#### Text Editing
    [ Ctrl ] + [ D ] : Delete line

#### Edit
    [ Alt ] + [ / ] : Content Assis

#### Source
    [ Ctrl ] + [ / ] : Toggle Comment
    [ Ctrl ] + [ Shift ] + [ O ] : Organize imports
    [ Ctrl ] + [ Shift ] + [ F ] : Format

#### Navigate
    [ Ctrl ] + [ T ] : Quick Type Hierarchy
    [ Ctrl ] + [ Alt ] + [ H ] : Open Call Hierarchy

#### Refactor
    [ Alt ] + [ Shift ] + [ R ] : Rename

#### Run/Debug
    [ F5 ] : Step into
    [ F6 ] : Step over
    [ F7 ] : Step return
    [ F8 ] : Resume
